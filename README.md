# Operation Blacksite — ML Experimental Division

Cold War sci-fi -teemainen portfolio-sivu machine learning -projekteille.

## Kuvaus

Sivu esittelee ML-projektit kylmän sodan aikaisen komentokeskuksen tunnelmassa. Jokainen projekti on oma "case file" josta pääsee demoon ja GitHub-repoon.

## Teknologiat

- HTML
- CSS (CSS-muuttujat, clamp(), animaatiot)
- Bootstrap 5.3
- JavaScript (vanilla)
- Google Fonts (Black Ops One, Teko, Roboto)

## Rakenne

- **Hero-osio** — Radar-animaatio ja sivun otsikko
- **Classified-palkki** — Tunnelmaa luova statusrivi
- **Case Files** — Projektikortit hover-efekteillä
- **Operational Tech Stack** — Interaktiivinen teknologialista joka päivittyy scramble-efektillä kortin hoverin mukaan
- **Footer** — Scramble-viestit ja elävä radiosignaali

## Ominaisuudet

- Responsiivinen (Bootstrap grid + clamp-fontit)
- Radar sonar-ping efekti hoverissa
- Tech stack scramble-efekti
- Mobiilissa sticky tech stack -palkki
- Hover-napit (Demo + GitHub) projektikorteissa
- Footer: vaihtuvat viestit + elävät koordinaatit

## Osa portfoliokokonaisuutta

Tämä sivu on yksi osa teemakohtaista portfolio-rakennetta:

- **Data Analysis** — Project Nexus
- **Machine Learning** — Operation Blacksite (tämä sivu)
- **Retro-futurismi** — tulossa
